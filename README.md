# My Best Friend v3.5.0 — Friend Design Sprint 1

**いつか 大きくなる、キミへの贈りもの**

## New in v3.5.0

- フレンド本体の静かな呼吸を実装
- 不規則で自然なまばたきを実装
- ときどき左右を見る小さな視線変化を追加
- 光と波紋をより穏やかな表現へ調整
- Homeの芽を大きくし、フレンドから離れた世界側へ固定
- 芽の影と揺れを自然に調整
- 雲と空の光をゆっくり動かすLiving Skyを調整
- 動きを減らす設定（prefers-reduced-motion）へ対応
- FriendStateの保存・全画面同期は維持

## Philosophy

ゲームを作るのではない。親友を育てる。

派手に動かすのではなく、静かな呼吸、まばたき、視線、風で「そこにいる」と感じられることを目指します。
