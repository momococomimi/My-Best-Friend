# My Best Friend v2.4.0 — Care & Mood

**いつか 大きくなる、キミへの贈りもの**

## Mission
世代を超えて受け継がれる親友を創る。

## New in v2.4.0
- Care画面を追加
- フレンドに気分（Mood）を追加
- 時間帯や再会タイミングでコメントが変化
- タッチやCareで気分が変化
- 気分に応じてAppearanceの色・光り方・表情が自動変化
- Homeのボタンを6個の均等サイズに整理
- Homeのフレンド・コメント・ボタンの余白を再調整

## Principle
Appearanceは手動着せ替えではなく、フレンドの心が自然に姿へにじみ出るもの。

**姿は変わる。親友は変わらない。**
